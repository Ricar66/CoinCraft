## 1. Primeiros Passos

Ao abrir o CoinCraft pela primeira vez, o sistema irá configurar automaticamente seu banco de dados local. Você não precisa de internet para utilizar o sistema, pois todos os dados ficam salvos apenas no seu computador.

### Licenciamento
Se esta for sua primeira execução, o sistema pode solicitar uma ativação. 
- **Modo Online:** Insira sua chave de licença se tiver conexão.
- **Modo Offline:** Siga as instruções para ativação via arquivo se não houver internet.

---

## 2. Dashboard

O **Dashboard** é sua central de comando. Nele você encontra:

- **Resumo do Mês:** Cartões mostrando total de Receitas, Despesas e o Saldo do período selecionado.
- **Filtro de Data:** No topo, você pode alterar o período de visualização (Mês atual, Ano, Personalizado).
- **Gráfico de Fluxo de Caixa:** Visualização da evolução de receitas vs despesas ao longo dos dias.
- **Despesas por Categoria:** Gráfico de rosca para entender onde você está gastando mais.
- **Metas:** Barra de progresso para suas metas de economia (configuráveis).

---

## 3. Gerenciando Contas

Acesse o menu **Contas** para cadastrar onde seu dinheiro está.

- **Adicionar Conta:** Clique em "+ Nova Conta".
  - Defina um nome (ex: "Nubank", "Carteira", "Itaú").
  - Informe o saldo inicial.
  - Escolha uma cor e ícone para facilitar identificação.
- **Editar/Excluir:** Selecione uma conta na lista para alterar dados ou removê-la.
  - *Nota:* Excluir uma conta pode afetar lançamentos vinculados a ela.

---

## 4. Categorias

Organize seus gastos criando **Categorias**. O sistema já vem com algumas padrão (Alimentação, Transporte, Lazer), mas você pode personalizar.

- **Hierarquia:** Crie categorias principais e subcategorias se desejar.
- **Limites:** Defina um teto de gastos mensal para cada categoria. O Dashboard irá alertá-lo se estiver próximo do limite.

---

## 5. Lançamentos (Receitas e Despesas)

Esta é a tela principal de operação diária.

- **Novo Lançamento:**
  1. Clique em "+ Novo".
  2. Escolha o Tipo (Receita, Despesa ou Transferência).
  3. Insira o Valor e a Data.
  4. Selecione a Conta e a Categoria.
  5. (Opcional) Adicione uma descrição e anexe um comprovante (imagem ou PDF).
- **Filtros:** Utilize a barra lateral para filtrar por Data, Conta ou Categoria.
- **Exportação:** Botões no topo permitem exportar a lista atual para Excel (CSV) ou PDF.

---

## 6. Lançamentos Recorrentes

Para contas fixas como Aluguel, Netflix, Salário:

1. Vá em **Recorrentes**.
2. Clique em "+ Novo Recorrente".
3. Preencha os dados como em um lançamento comum.
4. Defina a **Frequência** (Mensal, Semanal, Anual).
5. **Lançamento Automático:** Se marcado, o sistema lança sozinho no dia do vencimento. Se desmarcado, ele apenas notifica/aparece como pendente.

---

## 7. Importação de Dados

Para agilizar o cadastro, você pode importar extratos bancários.

- **Formatos Aceitos:** OFX (padrão bancário) e CSV.
- **Como Importar:**
  1. Clique em "Importar".
  2. Selecione o arquivo no seu computador.
  3. O sistema mostrará uma prévia dos lançamentos.
  4. Associe a uma Conta e clique em Confirmar.

---

## 8. Configurações

No menu **Configurações**, você pode:
- **Backup:** Criar uma cópia de segurança dos seus dados ou restaurar um backup anterior.
- **Tema:** Alternar entre modo Claro e Escuro.
- **Dados:** Limpar base de dados (Cuidado: irreversível!).
